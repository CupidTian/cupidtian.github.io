第一个JDBC

2019年11月8日21:01:59

第一步成功

记得是 Ctrl+alt+O 打开mk网页预览呀！

终于，我完成了第一步！

这个过程比较艰难

  • 第一个问题

Class.forName("com.mysql.jdbc.Driver");
此行语句出错，
提示过好像是版本问题
Loading class 'com.mysql.jdbc.Driver'. This is deprecated. The new driver class is 'com.mysql.cj.jdbc.Driver'.
查询后
Class.forName("com.mysql.cj.jdbc.Driver");
改正解决

  • 第二个问题

遇到了我的密码中有英文时候出现错误

然后花了好大功夫进行了修改密码的操作，
有三种修改mysql密码的操作，试了最后一种才不报错

访问了这个网页 解决办法
成功修改之后还是不成功
依然是这个问题
最后再进行错误查询，
需要在
conn = DriverManager.getConnection("jdbc:mysql://localhost:3306/myspj?serverTimezone=UTC", "root", "19990310");
那里补上一句很丑的话 ?serverTimezone=UTC"
这个好像是什么时区错误
然后成功解决

2019年11月8日21:53:41

executequery这个东西也成功了，现在可以独立的完成JDBC了

    解决eclipse，异常捕捉时自动生成的// TODO 自动生成的 catch 块
    解决办法
    决定休息一下下，继续战斗

  • 问题2

    少包的时候不能自动提示少的东西
    解决方案

期间有很小的问题，不至于说

  • 问题3
    最后调试的时候发现数据库名字写错了…多搞了一个小时….
