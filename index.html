定风波

三月七日，沙湖道中遇雨。雨具先去，同行皆狼狈，余独不觉，已而遂晴，故作此词。

莫听穿林打叶声，何妨吟啸且徐行。竹杖芒鞋轻胜马，谁怕？一蓑烟雨任平生。

料峭春风吹酒醒，微冷，山头斜照却相迎。回首向来萧瑟处，归去，也无风雨也无晴。

第一次静下心来品读这首词是高三毕业的时候，因为一好友把它写在了我的毕业留念册里。由于当时满心的离别与伤感，并未体会到词本身的意境，只是感觉到一种豁达开阔的气度，让人唇齿留香恋恋不忘。反而是好友娟秀的字迹让我印象更为深刻。

真正把这首词作为我的精神粮食，鼓舞我再次振作已经是十年后的事情了。

那是工作以来第一次遇到挫败，满脑子的委屈、不满、不甘和抱怨，自己陷在其中无法自拔，只能回到家嚎啕大哭以发泄情绪。吓坏了我妈，她从未见我如此失控过。而这首词的出现，就像一道光，东坡先生豁达洒脱的态度穿越千年，照亮了我那时黑暗的生活。让自己慢慢归于平静，从负面情绪中渐渐走出来。现在可以坦然面对那段过往，“回首向来萧瑟处，归去，也无风雨也无晴”。

工作四五年，谈不上“平步青云”，也算是顺风顺水，甚少波折。直到参与一个新项目。

个人还是蛮喜欢新项目的，什么都是全新的，需要调动全身每一个细胞去思考，去发现，趟出一条别人没走过来的路。我喜欢那种全身心投入到一件事情中去的感觉，那种忘我，让人感到陶醉。所以一直把这个新项目当做自己的孩子一般在用心呵护。因为涉及到五六个领域的事情需要同时处理和决策，包括招聘、售后、客服、产品、培训、风控以及APP开发等。工作时间根本不是九九六，基本上是七九六点五。在这个过程中承受的精神压力也是前所未有的，微信、邮件全部24h待命，无数次崩溃到在深夜抽泣。然而，从未想过要放弃，因为马云爸爸说“今天很残酷，明天跟残酷，后天很美好，但是绝大多数人死在明天晚上”，我想看看后天的美好。

然而，上天总是喜欢在你意想不到的时候跟你开玩笑。你不想走，却不得不走。

某天早上，照常早早去上班，发现工位布局都变了，原来的老板和同事搬到了另一个区域，我们部门被另外一个部门包围了。这个时候，部门内同事都陆续到来，大家都在犯嘀咕。但是依然没有一个人给我们一个解释或者说法。直到9:00开早会，直接宣布我们部门被别的部门合并了。

脑子里一万只苍蝇在嗡嗡叫，不愿接受却不得不接受，还要强作镇定。这么大的变动，没有任何事前沟通，就直接告知。我一直在心中问自己，这是为什么？明明已经那么努力，付出了那么多，之前大老板也很认可，为什么却发生这么猝不及防的变故。直到一个熟悉的同事，看我痛苦万分，私下告诉我，是因为我对大老板的方案提了一次不同的意见。

竟然是这样的原因，多么可笑，原来所有的努力都赶不上对老板的迎合。

只能，默默地选择离开……虽然已经离开，内心的痛苦却不减分毫。

感觉自己的天地已经灰暗，失去了所有的色彩。

“竹杖芒鞋轻胜马，谁怕？一蓑烟雨任平生”，这句词再次回响在脑海。

我翻箱倒柜，找到之前好友给我写的这首《定风波》，细细品来，突然内心澄明，有种顿悟的了然。

“莫听穿林打叶声，何妨吟啸且徐行”，刮风下雨时，焦急烦躁不如静心欣赏。不管心情如何糟糕，雨都不会停歇，那还不如“既来之则安之”，好好去欣赏沿途的风景，让雨水涤荡一下我们的灵魂。

当我们面对人生中的变故时，也是一样，不管心情怎样，该发生的都已经发生，重要的是如何面对这狂风暴雨，不被它淹没。等到风雨后见彩虹时，我们还有去拥抱的勇气和能力。

“竹杖芒鞋轻胜马，谁怕？一蓑烟雨任平生”，春风得意之人可以黄金貂裘荣华一生，落魄失意之徒也可以竹杖芒鞋潇洒一世。从黄金貂裘到竹杖芒鞋的转变会使大部分人无法接受，并非竹杖芒鞋的日子一定有多么难过，而是因为曾经拥有却又失去。要想解脱，必须放下！放下曾经的光鲜亮丽，放下鲜花和掌声，回归生活原本的淳朴与本真，多关注内在的所得，大不了从头再来。

“料峭春风吹酒醒，微冷，山头斜照却相迎”，人生有突如其来的横祸，也少不了意想不到的美好。特别喜欢一个同事的个性签名，“永远相信美好的事情即将发生”。不管现在是有冷风，还是有暴雨，深处苦难之时千万要坚持住。也许拐个弯，就能看见新的美好。我们需要做的不是伤心难过，自怨自艾，而是怀抱希望，乐观面对。

“回首向来萧瑟处，归去，也无风雨也无晴”，再大的委屈，再多的苦难，挺过去再回味，往往会对那段经历报以微笑，原来并不是那么大的事儿。还记得毕业之初，刚来北京，身上就三千多块钱。租不起房子，只能厚着脸皮挤同学那里。每天边找工作，边积极的帮她洗衣服、做饭，伺候周到。那时候也没想太多，总想着有了工作，领了工资就好了，再苦都硬挺着。现在回头望，一点也不觉得苦，反而很感恩有那样一段奋斗的经历，有人能在那个时候收留，就应该心怀感激。或许，很多的苦都是在当下把自己陷进去了，无法自拔，才感到苦。走过了，经历过了，它就是一段微甜的回忆。

现在离那段曾让我感到绝望的日子已经过去快1年了，每当心绪不宁愤愤不平时，总会拿笔静静地再写一遍这首词，让自己泰然处之。很感激自己能够快速地走出来，重新去发现生活的美好，也希望目前处在不愉快情境中的朋友们能够放下心中苦闷，拥抱生活的美好。

记得是 Ctrl+alt+O 打开mk网页预览呀！

这个过程比较艰难

  • 第一个问题

Class.forName("com.mysql.jdbc.Driver");
此行语句出错，
提示过好像是版本问题
Loading class 'com.mysql.jdbc.Driver'. This is deprecated. The new driver class is 'com.mysql.cj.jdbc.Driver'.
查询后
Class.forName("com.mysql.cj.jdbc.Driver");
改正解决

  • 第二个问题

遇到了我的密码中有英文时候出现错误

然后花了好大功夫进行了修改密码的操作，
有三种修改mysql密码的操作，试了最后一种才不报错

访问了这个网页 解决办法
成功修改之后还是不成功
依然是这个问题
最后再进行错误查询，
需要在
conn = DriverManager.getConnection("jdbc:mysql://localhost:3306/myspj?serverTimezone=UTC", "root", "19990310");
那里补上一句很丑的话 ?serverTimezone=UTC"
这个好像是什么时区错误
然后成功解决
